# 🏎️ Roman Kim — Personal Portfolio Website

> A dark, high-impact personal portfolio inspired by the Oracle Red Bull Racing aesthetic.
> Built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks, no dependencies.

---

## 📸 Preview

> Open `index.html` in any browser to view the site locally.

---

## 🚀 Features

- **Animated hero section** with staggered text reveal and ghost watermark
- **Scrolling red ticker** (inspired by the Red Bull Racing site)
- **Sticky navbar** that blurs on scroll
- **Scroll-reveal animations** on every section
- **Fully responsive** layout for mobile and desktop
- **Clickable contact links** — Email, GitHub, LinkedIn, TikTok, Phone

## 🛠️ How to Run

No build tools or installations needed.

```bash
# Option 1 — just open the file
open index.html

# Option 2 — serve locally with Python
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🧱 Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **JavaScript (ES6)** — IntersectionObserver for scroll reveals, navbar scroll effect
